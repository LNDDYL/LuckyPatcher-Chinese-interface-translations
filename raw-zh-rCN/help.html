分类说明：
---------------------------------
绿色 - 此应用程序有很大机会被破解；
黄色 - 可在/sdcard/LuckyPatcher/找到自定义破解；
青色 - 发现该应用程序有谷歌广告；
洋红 - 此应用程序已添加至启动列表；
紫色 - 支持谷歌内购的应用程序；
红色 - 此应用程序没有机会被破解，该异常是因为应用程序拥有两部分（应用程序和专业版密钥）。在这种情况下，需要同时破解两个或其中一个应用程序；
橙色 - 此为系统应用程序，对系统应用程序执行任何操作时，请务必谨慎，它可能对应用程序或操作系统造成严重的不良后果。
启动列表 - 开机后自动应用破解的应用程序列表。此列表仅在应用程序需要修复库文件 “.so” 时才使用。但不存在自定义破解，除非用于已不再使用的旧版本钛备份。

四叶草 - 显示应用程序破解模式更改后的状态；
星标 - 意味着应用程序的虚拟机缓存已被修复（已创建 ODEX 更改），相反则意味着程序将保持当前的状态，直至应用程序被再次更新或删除。如果此 ODEX 使用 “移除 ODEX 更改” 来移除，那应用程序将会完全恢复到被应用破解前的状态。当您更新应用程序时，问题就会出现：电子市场将会提示空间不足，此时，运行幸运破解器并关闭，再尝试安装该应用程序。如果这样的错误出现在已卸载的应用程序中，请选择 “故障排除 -> 移除修复和备份（更新程序）” - 所有修复的应用程序虚拟机缓存将会被删除！

上下文菜单：
---------------------------------
移除授权验证（自动模式）
- 尝试移除应用程序的授权验证。将会使用小部分的破解模块来进行破解以确保应用程序的最大稳定性。
移除授权验证（自动模式(逆向破解)）
- 此模式是 “自动模式” 的逆向破解模式。如果 “自动模式” 破解后没有效果，可以尝试使用此项进行破解。
移除授权验证（极限自动模式）
- 尝试移除应用程序的授权验证。将会使用所有的破解模块来进行破解，但破解后的应用程序不稳定，在使用过程中会产生更多的错误。
移除授权验证（手动）
- 尝试移除应用程序的授权验证。将会手动选择破解模式来进行强力移除应用程序的授权验证。
移除授权验证（选择破解模式）
- 您需要选择一组破解模块来对应用程序进行破解。
移除谷歌广告！
- 尝试移除应用程序自带的谷歌广告。有时候应用程序会处于未发现任何情况的状态，您需要从未找到广告标签的应用程序中进行移除谷歌广告。
移除 ODEX 更改
- 移除 ODEX 文件及包含其更改，将应用程序恢复到初始状态。
更改权限和活动项
- 移除应用程序请求系统的权限。请谨慎对此项进行操作，由于在某些设备中更改权限后会导致无限重启或其他不良后果，因此，在您尝试更改权限前建议先对系统进行备份。
创建修改的 APK 版本
- 基于可用的破解来创建一个 APK 安装包。请注意：重建的 APK 文件对于已安装的程序被应用破解的效果并不同，由于应用破解的程序可以很简易的测试自身的更改并使用正确，但安装重建的 APK 文件使用效果并非一样。
重启时应用破解
- 此项仅在应用程序需要修复库文件 “.so” 时才使用，但不存在自定义破解，除非用于已不再使用的旧版本钛备份。如果应用程序在重启后应用了破解出现不稳定，它就需要添加不超过一个自定义破解。如果是应用程序使用时必需的，它将会被写入一个自定义破解。

手动破解器！
- 调试应用程序的破解器。
备份！
- 备份 APK 文件到 /sdcard/LuckyPatcher/Backup/ 文件夹。
恢复！
- 从 /sdcard/LuckyPatcher/Backup/ 文件夹恢复备份的 APK 文件。


复制更改到虚拟机缓存！
- 在默认模式下的所有更改将会保留 ODEX 文件。但某些 ODEX 文件却不能工作于相应的应用程序（例如：当移动应用程序到外部 SD 卡时），因此，复制更改到虚拟机缓存就可以解决此问题。但是，如果您只想重新安装该应用程序，您可以直接恢复它。
备份重新安装的 APK 文件！
- 复制 APK 文件到 SD 卡的 “LuckyPatcher” 数据文件夹。复制成功后可以从这个 APK 文件重新安装（恢复）相应的应用程序。
删除虚拟机缓存！
- 删除此应用程序在虚拟机缓存创建后的 ODEX 文件以释放空间。如果 ODEX 文件不能工作于该应用程序，请重新安装此应用程序。

破解说明：
---------------------------------
移除授权验证：
- 只能用于开发者签名的原始 APK 文件
- 长按应用程序并选择 “移除授权验证”
- 选择 “自动模式” 并连接网络测试应用程序
- 若破解没有效果，再选择 “极限自动模式”
- 如果您够幸运，应用程序将会被破解
- 点击 “移除 ODEX 更改” 可恢复应用程序到初始状态

移除谷歌广告：
- 长按应用程序并选择 “移除谷歌广告”
- 运行并连接网络测试应用程序
- 如果您够幸运，谷歌广告将会被移除
- 点击 “移除 ODEX 更改” 可恢复应用程序到初始状态

自定义破解：
- 长按应用程序并选择 “自定义破解”
- 运行并连接网络测试应用程序
- 如果您够幸运，应用程序将会被破解
- 点击 “移除 ODEX 更改” 可恢复应用程序到初始状态

更改权限和活动项：
- 长按应用程序并选择 “更改权限和活动项”
- 单击不要的权限可停用 （红色）
- 单击需要的权限可启用 （绿色）
- 最后必须点击 “立即重启应用更改”。

创建修改的 APK 版本：
- 长按应用程序并选择 “创建修改的 APK 版本”
- 再根据需要选择要创建的 APK 版本
- 已修改的安装文件将会保存在 /sdcard/LuckPatcher/Modified/ 文件夹
- 请注意：重建的 APK 文件对于已安装的程序被应用破解的效果并不同，由于应用破解的程序可以很简易的测试自身的更改并使用正确，但安装重建的 APK 文件使用效果并非一样。

备注：当应用破解时，幸运破解器会关闭其应用程序，为了获取新的更改必需如此。但有时您可能需要重新启动设备或使用第三方工具来终止应用程序的进程。
